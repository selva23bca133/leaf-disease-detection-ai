# Leaf Disease Detection AI

This project detects plant leaf diseases using a Convolutional Neural Network (CNN) model built with TensorFlow and a Streamlit web interface.

## Features
- Upload leaf image
- AI detects healthy or diseased leaf
- Displays model confidence
- Provides treatment recommendation

## Technologies
- Python
- TensorFlow / Keras
- Streamlit
- NumPy
- OpenCV

## Run the project
pip install -r requirements.txt
streamlit run app.py
